Mis proyectos sobre métodos numéricos en R Markdown

Proyecto 1: Métodos para calcular raíces de funciones

Este proyecto implementa cuatro métodos numéricos en R para encontrar raíces (ceros) de funciones reales. Incluye comparaciones de convergencia, análisis de errores y visualización del comportamiento de cada algoritmo.

Proyecto 2: Métodos numéricos para integración y aproximación

Este proyecto implementa cuatro métodos numéricos fundamentales en R para aproximar funciones y calcular integrales definidas. Incluye visualizaciones y comparaciones de precisión entre los diferentes métodos.
